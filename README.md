# VSCode YAML extension bug report

## How to reproduce

1. Open `schema.yml` in VSCode.
2. Confirm that 2 schemas (`schema.yml` itself and the standard JSON schema) are activated.

## What it should be

1. Open `schema.yml` in VSCode.
1. Only the standard JSON schema is activated.
